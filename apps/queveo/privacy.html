QuéVeo · Última actualización: marzo de 2026

QuéVeo es una aplicación diseñada con la privacidad como principio. Tus preferencias, historial y valoraciones se almacenan exclusivamente en tu dispositivo. No existe registro de usuario ni inicio de sesión.

1. Quién es el responsable

Esta aplicación es desarrollada y mantenida por Clevora Labs de forma independiente. Para cualquier consulta relacionada con la privacidad puedes contactar en: privacy@clevoralabs.com

2. Qué datos se almacenan y dónde

QuéVeo almacena la siguiente información únicamente en tu dispositivo, usando la base de datos local de la aplicación:

  • Tus preferencias de géneros cinematográficos
  • Las plataformas de streaming que has indicado
  • Tu lista de títulos guardados (watchlist)
  • Las valoraciones que has dado a películas y series
  • El historial de recomendaciones generadas
  • Preferencias de duración y tipo de contenido
Estos datos no se sincronizan con ningún servidor externo y desaparecen si desinstalas la aplicación.

3. Servicios externos utilizados

Para ofrecer sus servicios, QuéVeo se conecta a los siguientes servicios de terceros:

  • The Movie Database (TMDb): Se consulta su API pública para obtener información sobre películas y series. Solo se envía la consulta de búsqueda, nunca datos personales.
  • Google Gemini API: Se envía un perfil de preferencias anonimizado para generar recomendaciones personalizadas. No se incluyen datos identificativos.
  • Firebase Analytics (Google): Recopila datos de uso anónimos como pantallas visitadas, eventos de interacción, idioma, versión de SO y país.
  • Firebase Crashlytics (Google): En caso de error, recopila información técnica sobre el fallo para permitirnos corregirlo, siempre de manera anónima.
  • Google AdMob (Publicidad): Utilizamos Google AdMob para mostrar anuncios. Para ello, el SDK de AdMob recopila y utiliza el ID de publicidad de Android (AAID). Este identificador permite a Google mostrar anuncios relevantes para el usuario, limitar el número de veces que se ve un mismo anuncio, combatir el fraude y abuso y realizar mediciones de rendimiento de los anuncios.

4. Datos que recopilamos

  • No recopilamos nombre, email ni ningún dato de identificación personal.
  • No existe registro ni inicio de sesión.
  • ID de Publicidad: Se utiliza exclusivamente para la gestión de anuncios a través de AdMob.
  • Datos de uso: Información técnica sobre el dispositivo y navegación anónima para mejora del servicio.

5. Control y Publicidad Personalizada

Como usuario/a, tienes el control total sobre los anuncios:

  • Desactivación de anuncios personalizados: Puedes restablecer tu ID de publicidad o desactivar la personalización de anuncios en los ajustes de tu dispositivo: Ajustes > Google > Anuncios.
  • Más información: Puedes consultar cómo utiliza Google la información en https://policies.google.com/technologies/partner-sites

6. Base legal para el tratamiento de datos (RGPD)

El tratamiento de datos para análisis y publicidad se basa en el interés legítimo para el mantenimiento económico y la mejora técnica de la aplicación, garantizando siempre la opción de "opt-out" del usuario a través de los ajustes de su terminal.

7. Menores de edad

QuéVeo no está dirigida específicamente a menores de 13 años. Los anuncios mostrados a través de AdMob cumplen con las políticas de contenido de Google Play para su categoría.

8. Tus derechos

Al almacenarse los datos localmente, puedes eliminarlos desinstalando la aplicación. Para cualquier duda sobre el uso del ID de publicidad, contacta con nosotros.

9. Contacto

Para cualquier duda o pregunta puedes ponerte en contacto con nostros en la siguiente dirección: privacy@clevoralabs.com
